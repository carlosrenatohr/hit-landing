## PARTE A — RESCATE DE LA RIFA (URGENTE: 7 días restantes)

### A.1 — Diagnóstico: por qué casi no hay resultados

El algoritmo de Instagram muestra una publicación orgánica al **2-5% de los seguidores**. Si la cuenta tiene 500 seguidores, eso es 10-25 personas que vieron el post. Sin distribución pagada, una rifa no existe.

Tres causas probables en orden de impacto:
1. Poco alcance orgánico — cuenta pequeña, sin boost pagado
2. El post no para el scroll en los primeros 2 segundos
3. Sin recordatorios diarios que mantengan el momentum

### B.1 — Meta Pixel (Renato — 1 hora)

Conecta el comportamiento en el sitio con las campañas de Instagram/Facebook. Sin él, los ads van "a ciegas". El sitio ya tiene GTM activo (ID: `GTM-K55VC9JZ`) — el pixel se agrega desde adentro de GTM sin tocar el código.

**Pasos:**
1. En Meta Business Suite → Administrador de eventos → Crear píxel → copiar el ID (formato: 15 dígitos)
2. Ir a tagmanager.google.com → Contenedor GTM-K55VC9JZ
3. Etiquetas → Nueva → buscar "Facebook Pixel" en plantillas de la comunidad
4. Pegar el ID del Pixel
5. Activador: All Pages → Guardar
6. Publicar el contenedor

**Nota de seguridad:** Si aparecen errores de CSP en la consola del navegador después de activarlo, agregar estas dos líneas al `public/_headers` en la sección `connect-src`:
```
https://connect.facebook.net
https://www.facebook.com/tr/
```

**Eventos a agregar después (cuando la calculadora esté lista):**
- `ViewContent` → cuando alguien abre la calculadora

### B.2 — Google Analytics 4 (Renato — 30 minutos)

GTM ya está activo. Hay que verificar que GA4 recibe datos reales.

1. analytics.google.com → Crear propiedad GA4 si no existe
2. Copiar ID de medición (formato: `G-XXXXXXXXXX`)
3. GTM → Etiquetas → Nueva → "Google Analytics: Configuración de GA4"
4. Pegar ID → Activador: All Pages → Publicar
5. Verificar: GA4 → Informes → Tiempo real → abrir el sitio en otra pestaña → debe aparecer 1 usuario activo

---

### B.3 — Google Search Console (Renato — 20 minutos)

Le dice a Google que el sitio existe. Sin esto, puede tardar semanas en indexarse solo.

1. search.google.com/search-console → Agregar propiedad → URL: `https://hit-cargo.com`
2. Verificar via Google Tag Manager (el método más fácil, ya que GTM está activo)
3. Una vez verificado: Sitemaps → ingresar `sitemap.xml` → Enviar
4. Astro genera el sitemap automáticamente si `@astrojs/sitemap` está instalado — verificar que aparezca en `/sitemap.xml`

**Revisar semanalmente el primer mes:**
- Cobertura (páginas indexadas vs. con errores)
- Búsquedas que generan impresiones
